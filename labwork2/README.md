# Лабораторная работа №2: Численное решение нелинейных уравнений и систем

Данный проект представляет собой клиент-серверное веб-приложение для численного решения нелинейных уравнений и систем нелинейных уравнений. Выполнен в рамках курса «Вычислительная математика» (ИТМО).

## Особенности
- **Одиночные уравнения:** Решение методами хорд, Ньютона и методом простой итерации (МПИ).
- **Системы уравнений:** Решение методом простой итерации (МПИ).
- **Анализ сходимости:** 
  - Проверка наличия корня на отрезке и монотонности функции.
  - Расчет коэффициента сходимости $q$ и параметра $\lambda$ для одиночных уравнений.
  - Проверка достаточного условия сходимости для систем (вычисление норм матрицы Якоби).
- **Продвинутая визуализация:** 
  - Динамическое построение графиков функций и изоляция корней (с использованием `Recharts`).
  - Построение неявных функций и линий уровня (contour plots) с помощью `Matplotlib` на стороне сервера для систем уравнений.
- **Подробный вывод:** Отображение таблицы с пошаговыми итерациями, вычислением невязок и погрешностей.

## Стек технологий
- **Backend:** Python 3.10+, FastAPI, NumPy, Matplotlib, Pydantic.
- **Frontend:** React, JavaScript, Vite, Axios, Recharts.

## Структура проекта
- `/app` — серверная часть (API, математическая логика, генерация графиков).
- `/frontend` — клиентская часть (пользовательский интерфейс, формы ввода, отрисовка таблиц и графиков).

## Как запустить проект

### 1. Запуск Backend (Серверная часть)
Откройте терминал в корневой папке бэкенда (там, где находится файл `requirements.txt`) и выполните следующие команды:

```bash
# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Запуск сервера
uvicorn app.main:app --reload
```

API будет доступно по адресу: http://127.0.0.1:8000  
Документация к API (Swagger): http://127.0.0.1:8000/docs

### 2. Запуск Frontend (Клиентская часть)
Откройте новый терминал в папке `frontend` и выполните команды:
```bash
# Установка зависимостей Node.js
npm install

# Запуск сервера разработки
npm run dev
```
После этого в консоли появится локальная ссылка (обычно `http://localhost:5173`), по которой можно открыть интерфейс приложения в браузере.

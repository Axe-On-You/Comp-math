# Лабораторная работа №1: Решение СЛАУ (Метод Гаусса)

Данный проект представляет собой клиент-серверное веб-приложение для решения Систем Линейных Алгебраических Уравнений (СЛАУ) с помощью обычного метода Гаусса. 

## Особенности
- Ввод размерности матрицы ($n \le 20$) и коэффициентов как вручную через UI, так и загрузкой из файла.
- Вывод верхнетреугольной матрицы (включая преобразованный вектор свободных членов $B$).
- Вычисление определителя системы по диагонали треугольной матрицы.
- Нахождение вектора неизвестных и вычисление вектора невязок.
- Сравнение результатов с эталонной библиотекой `NumPy`.

## Стек технологий
- **Backend:** Python 3.14, FastAPI, NumPy, Pydantic.
- **Frontend:** React, TypeScript, Vite, Axios.

## Структура проекта
- `/backend` — серверная часть (API, вычислительная логика).
- `/frontend` — клиентская часть (пользовательский интерфейс).

## Как запустить проект

### 1. Запуск Backend (Серверная часть)
Откройте терминал в папке `backend` и выполните следующие команды:
```bash
# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate

# Установка зависимостей
pip install requirements.txt

# Запуск сервера
uvicorn main:app --reload
```
API будет доступно по адресу: http://127.0.0.1:8000  
Документация к API (Swagger): http://127.0.0.1:8000/docs

### 2. Запуск Frontend (Клиентская часть)
Откройте новый терминал в папке `frontend` и выполните команды:
```bash
# Установка зависимостей Node.js
npm install

# Запуск сервера разработки
npm run dev
```
После этого в консоли появится локальная ссылка (обычно `http://localhost:5173`), по которой можно открыть интерфейс приложения в браузере.
